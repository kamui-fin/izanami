# Izanami

A general-purpose utility bot for the [Japan Zone](https://disboard.org/server/732631790191378453) discord server.

Features include:

-   JLPT quiz barrier
-   Retrive information and suggest recommendations for:
    -   Anime
    -   Manga
    -   Drama
    -   Light Novels
    -   Visual Novels
-   Event scheduling
-   Welcome messages
-   Daily news
-   Disboard bump notifications
